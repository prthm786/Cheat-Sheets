# Windows Shortcuts

1. Win I -> Settings

2. Win E -> File Explorer

3. Win W -> Widgets Panels

4. Win X or right click on Start Menu -> Power Menu

5. Win 0-9 -> Open Pinned Apps (apps in dock/taskbar)

6. Win A -> Quick Settings

7. Win N -> Notifications & Calendar

8. Win D -> Toggle Desktop

9. Win Home -> Minimize all windows accept current

10. Alt f4 -> Close Program

11. Alt Tab -> Switch Apps

12. Win Arrow keys -> Snap Windows

13. Win Up Arrow then Win Left Arrow -> left quarter of  
    screen
14. Win Up Arrow then Win Right Arrow -> right quarter of  
    screen
15. Win Z -> Snap Layouts

16. Win X U U -> Quick Shutdown

17. Win X U R -> Restart

18. Win H -> Voice Dictation

19. Win V -> Win Clipboard History

20. Ctrl Z -> Undo

21. Ctrl Y -> Redo

22. Ctrl A -> Select All

23. Ctrl shift V -> Paste Without Formatting

24. Ctrl click -> Select Multiple items (in file explorer)

25. Shift click -> Select Range of items (in file explorer)

26. Ctrl L -> Edit Path (in file explorer)

27. Ctrl shift Esc or Win X T -> Open Task Manager

28. Win ; or Win . -> Emoji

29. Win R -> Run Utitlity

30. Win Shift S or PrtSrn -> Screenshot

31. Win Shift R -> Screen Recording

32. Win Ctrl S -> Voice Commands

33. Win Ctrl L -> Live Captions

34. Win Alt S -> Sticky Notes

35. Ctrl W -> Close Tab

36. Ctrl A -> Select all

37. Win Tab -> Open Task View

38. Tab and Shift+Tab -> Move backward and forward through options

39. Win L -> Lock Computer

40. Ctrl Alt Del -> Shutdown/Logout

41. Ctrl alt arrows -> change orientation

42. Alt arrow keys ->

43. Ctrl Home -> Start of page

44. Ctrl End -> End of page

45. Home -> Start of line

46. End -> End of line

47. Ctrl Left/Right Arrow Keys -> Move Cursor word by word

48. F2 -> Rename

49. Win G -> Game Bar

50. Alt Enter -> Properties of File/Folder

51. Ctrl Shift N -> New folder

52. Win C -> Copilot

53. Win Q or Win Click -> Click To Do

54. Win Ctrl Q -> Quick Assist

55. Ctrl K -> Casting

56. F5 -> Refresh

57. Ctrl win D -> New Virtual Desktop

58. Ctrl win ->/<- -> Switch Desktops

59. Ctrl Del -> Delete Word to Right

60. Ctrl Backspace -> Delete Word to Left

61. Shift Del -> Permanently Delete File
