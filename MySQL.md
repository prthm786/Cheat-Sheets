# MySQL Commands 

Listing Databases
```
  $ SHOW DATABASES;
```

Creating a Database
```
  $ CREATE DATABASE <db_name>
```

Using Database 
```
  $ USE <db_name>;
```

Creating a Table 
```
  $ CREATE TABLE <Table_name> (
    <column_name> <datatype>,
    <column_name> <datatype>
    );
```
